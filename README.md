# Web Shop for learning to create react apps

## Tech used

### Frontend

✔️   React<br/>
✔️   redux<br/>
✔️   redux saga<br/>
✔️   reselect (memoization)<br/>
✔️   Styled Components<br/>

### Backend

✔️   firebase (database and user login)<br/>
✔️   express (server)<br/>
✔️   Heroku (hosting)<br/>
✔️   Stripe (handling payments)<br/>
